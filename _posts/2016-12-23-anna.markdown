Программа предназначена для построения аналитических таблиц, которые используются для
доказательства общезначимости логической формулы.

## Запуск
{% highlight bash %}
python console.py "((A->B)->((A->(B->C))->(A->C)))<->A" > result.html
{% endhighlight %}
